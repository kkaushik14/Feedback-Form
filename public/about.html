About This Web App

Movie Feedback is a modern, full-stack SaaS web application for sharing and discovering movie reviews and recommendations.
Users can sign up, log in, and submit feedback for any movie, as well as browse, search, and interact with reviews from others.

  • Features: User authentication (sign up/login), movie feedback submission, review browsing and search, movie recommendations, responsive navigation and footers, professional SaaS-style UI, and developer contact section.
  • Frontend: HTML5, Tailwind CSS, Fira Mono font, JavaScript (ES6+), Bootstrap Icons, responsive design, and interactive UI/UX with card transitions.
  • Backend: Node.js, Express.js, JWT authentication, cookie-parser, bcryptjs for password hashing, and modular route handling.
  • Database: MongoDB (Mongoose ODM), storing users, feedback, and review data securely.
  • Middlewares & Dependencies: express.json, express.static, dotenv, cookie-parser, bcryptjs, jsonwebtoken, mongoose, and CORS.

Whether you want to share your thoughts on a film, find top-rated movies, or connect with other movie lovers, Movie Feedback makes it easy and enjoyable.
